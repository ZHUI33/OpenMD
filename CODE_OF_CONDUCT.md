# Code of Conduct

OpenMD 社区致力于提供友好、安全、尊重且不骚扰的协作环境，不因经验、身份、背景或能力而区别对待。

参与者应当：

- 使用包容、专业的语言，善意解释技术分歧。
- 尊重他人的时间、隐私、边界和不同观点。
- 聚焦对代码、设计和事实的具体反馈，不进行人身攻击。
- 不骚扰、威胁、歧视、公开他人隐私信息或实施其他不当行为。

维护者可以删除不当内容、关闭讨论或限制持续违反规范者的参与。需要报告问题时，请通过仓库维护者可见的
私密渠道或 GitHub Security Advisory 联系，避免在公开 Issue 中披露敏感个人信息。维护者会尽量保密、
公正地处理报告。

本规范适用于仓库、Issue、Pull Request、发布渠道以及代表 OpenMD 参与的其他社区空间。
